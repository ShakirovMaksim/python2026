6. **Полное решение**  
```python
def mean_abs_deviation(*scores):
    if not scores:
        raise ValueError("Необходимо передать хотя бы одно значение.")
    
    avg = sum(scores) / len(scores)
    deviations = [round(abs(score - avg), 1) for score in scores]
    return tuple(deviations)
```
