### 6. Полное решение для `create_report`
```python
def create_report(task_list, sort_by_desc=False, header="Отчет по задачам"):
    work_list = list(task_list)
    if sort_by_desc:
        work_list.sort(key=lambda task: task["description"])
    report_text = f"{header}\n"
    for i, task in enumerate(work_list, 1):
        tags_str = ", ".join(task["tags"])
        report_text += f"{i}. {task['description']} [{tags_str}]\n"
    return report_text
```
