#### 1. Формулировка задания
**Цель:** Найти общие и уникальные тренажёры между утренними (`set_a`) и вечерними (`set_b`) бронированиями в фитнес-клубе.  
**Входные данные:** Два множества строк (названия тренажёров), например:  
```python
set_a = {'bike', 'press', 'row'}
set_b = {'bike', 'boxing', 'stretch'}
```  
**Ожидаемый вывод:**  
- Множество общих тренажёров.  
- Множество тренажёров, уникальных для вечера (или утра, в зависимости от логики).  
**Ключевые ограничения:**  
- Использовать операции над множествами (`&`, `-`, `|`).  
- Не преобразовывать множества в списки без необходимости.  
- Выводить осмысленные подписи для результатов.  

---

#### 2. Результаты проверки

```python
set_a = {'bike', 'press', 'row'}

```  
**Тестирование:**
