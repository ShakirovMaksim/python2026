### Отчёт по проверке задания 31.2

---

#### 1. Формулировка задания
**Цель**: Суммировать количество коробок по бригадам из списка смен (`shifts`), безопасно удалить команды из списка `finished` и вывести обновлённый словарь.  
**Ключевые требования**:
- Накопление значений через `get()` или `setdefault()`
- Удаление записей через `pop(key, default)` для избежания `KeyError`
- Вывод обновлённого словаря после удаления
- Обработка случаев, когда ключи из `finished` отсутствуют в словаре

**Пример входных данных**:
```python
shifts = [('team-A', 40), ('team-B', 32), ('team-A', 15), ('team-C', 20)]
finished = ['team-B']
```

---

#### 2. Результаты проверки
**Файл**: `task.py`  
**Код студента**:
```python
boxes = {}
for team, count in shifts:
    boxes[team] = boxes.get(team, 0) + count  # Накопление значений
for team in finished:
    boxes.pop(team, 0)  # Безопасное удаление
    print(f'Закончила раньше всех бригада: {team}')
print(boxes)
```

**Тестирование**:
- **Стандартный случай**:  
  Входные данные: `shifts` и `finished` из примера.  
  Результат: `{'team-A': 55, 'team-C': 20}` (корректно).
- **Пустой `finished`**:  
  Входные данные: `finished = []`.  
  Результат: вывод исходного словаря без изменений (корректно).
- **Несуществующая команда в `finished`**:  
  Входные данные: `finished = ['team-X']`.  
  Результат: отсутствие ошибок, вывод исходного словаря (корректно).
