### Задание 31.1. Основы Python. Работа со строками и списками

### Описание
Напиши программу, которая получает на вход логин пользователя и проверяет его корректность по простым правилам.

Требования:
- убрать пробелы по краям строки
- проверить, что длина логина от 3 до 20 символов
- логин должен состоять только из букв латинского алфавита и цифр
- вывести сообщение о корректности или список нарушенных правил

### Пример входных данных
```
  max_2026  
```

### Пример выходных данных
```
Некорректный логин
Нельзя использовать символ: _
```

### Критерии успешного выполнения
- логин очищается от пробелов по краям
- длина проверяется корректно
- каждый символ логина проверяется по условию
- выводятся понятные сообщения о нарушениях

### Подсказки
- используй `strip` для очистки
- для проверки символов можно использовать методы `isalnum` и проверки диапазонов

### Частые ошибки и ограничения
- не останавливайся на первой ошибке . покажи все нарушения
- не используй регулярные выражения . сделай проверки вручную
