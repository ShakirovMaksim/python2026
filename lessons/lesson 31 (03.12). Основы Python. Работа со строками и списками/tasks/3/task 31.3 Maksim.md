### Задание 31.3. Основы Python. Работа со строками и списками

### Описание
Напиши программу, которая помогает работать со списком задач на день.

Требования:
- прочитать строку с задачами, разделенными точкой с запятой `;`
- разбить ее на список задач
- удалить пробелы по краям каждой задачи
- удалить пустые элементы
- вывести количество задач и красиво отформатировать их по строкам с нумерацией

### Пример входных данных
```
  учить Python ;  решить задачи ; ; посмотреть разбор  
```

### Пример выходных данных
```
Всего задач: 3
1. учить Python
2. решить задачи
3. посмотреть разбор
```

### Критерии успешного выполнения
- исходная строка корректно разбита по `;`
- пустые и пробельные элементы удалены
- задачи выведены с нумерацией и без лишних пробелов

### Подсказки
- используй `split(';')` и `strip()`
- применяй генератор списка для фильтрации пустых строк

### Частые ошибки и ограничения
- не забывай удалять полностью пустые строки после `strip`
- не используй внешние библиотеки . хватает базового Python
