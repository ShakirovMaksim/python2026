Задание 2. Основы Python. Переменные и типы данных

### Описание
Сделай скрипт, который читает название проекта, бюджет и срок выполнения в днях. Приведи бюджет к float, срок к int. Выведи итоговый отчёт с указанием введенных значений, их типов и расчетом бюджета на один день.

### Пример входных данных
```
Site redesign
45000.0
15
```

### Пример выходных данных
```
Проект: Site redesign (str)
Бюджет: 45000.0 (float)
Срок: 15 (int)
Бюджет в день: 3000.0 (float)
```

### Критерии успешного выполнения
- Данные читаются через input() в порядке: проект, бюджет, срок.
- Бюджет преобразуется к float, срок к int.
- Рассчитывается стоимость одного дня без ошибок деления.
- Вывод содержит подписи, значения и типы.
- Программа корректно завершается при нормальном вводе.

### Подсказки
- Проверь, что преобразование выполняется до расчетов.
- Используй переменную для результата деления и выводи ее отдельно.
- Применяй f-строки для форматирования.

### Частые ошибки и ограничения
- Не путай порядок чтения, иначе преобразование завершится ошибкой.
- Избегай деления на ноль. требуй положительный срок.
- Не округляй результат вручную. оставь поведение float по умолчанию.
