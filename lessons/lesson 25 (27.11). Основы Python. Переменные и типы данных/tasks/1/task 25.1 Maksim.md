Задание 1. Основы Python. Переменные и типы данных

### Описание
Создай программу, которая получает от пользователя имя, фамилию и возраст. Сохрани имя и фамилию как строки, возраст приведи к int. Выведи итог в формате анкеты с указанием типа каждого значения.

### Пример входных данных
```
Maksim
Petrov
18
```

### Пример выходных данных
```
Имя: Maksim (str)
Фамилия: Petrov (str)
Возраст: 18 (int)
```

### Критерии успешного выполнения
- Используются три вызова input() в указанной последовательности.
- Возраст преобразуется к целому числу до вывода.
- Строки выводятся без изменений и с подписями.
- В итоговом выводе присутствует тип каждого значения.
- Скрипт завершается без ошибок при корректном вводе.

### Подсказки
- Храни каждое значение в отдельной переменной с коротким именем.
- Функцию type(...)__name__ можно сохранить в переменной для повторного использования.
- Для аккуратного вывода применяй f-строки.

### Частые ошибки и ограничения
- Не меняй порядок чтения input(), иначе преобразование к int даст ошибку.
- Не добавляй лишний текст в вывод кроме требуемых строк.
- Не пытайся преобразовать строковые значения к числу.
