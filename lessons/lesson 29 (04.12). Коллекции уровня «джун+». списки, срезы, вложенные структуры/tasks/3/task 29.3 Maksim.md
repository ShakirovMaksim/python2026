# Задание 29.3. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Мутация и методы списков. Отрабатываем добавление новых элементов, вставку по точному индексу и безопасное удаление без создания копий списка.

## Условие задачи
Ты собираешь плейлист для интервальной тренировки и должен быстро менять порядок треков:
1. Создай список базовых треков по фазам тренировки.
2. Добавь в конец два трека для заминки, полученных из отдельного списка, с помощью `extend`.
3. Вставь мотивирующий джингл в позицию №1 перед основной частью.
4. Удали трек, который не подходит по темпу, по значению.
5. Перемести последний трек в середину списка, используя комбинацию `pop` и `insert`.
6. Выведи итоговый порядок с нумерацией, чтобы свериться перед занятием.

## Пример входных данных
```python
playlist = ["разминка", "спринт", "поддержание", "ускорение"]
cooldown = ["растяжка", "медитация"]
```

## Пример выходных данных
```
1. джингл
2. разминка
3. спринт
4. поддержание
5. ускорение
6. растяжка
```

## Критерии успешного выполнения
- Использованы минимум три разных метода мутации
- Вставка и удаление проводятся по корректным индексам и значениям
- Перемещение выполнено без ручного обмена переменных
- Итоговый список напечатан после всех правок

## Подсказки
- После `extend` список сразу содержит элементы из `cooldown`
- Индекс 1 вставляет элемент перед второй позицией
- Чтобы перенести конец в середину, сохрани `item = playlist.pop()` и вставь по индексу `len(playlist)//2`

## Частые ошибки и ограничения
- Не вызывай `remove`, если значение не встречается
- Избегай изменения списка во время цикла `for`
- Следи за делением `len(playlist)//2`, чтобы индекс не выходил за пределы
- Не создавай новый список. изменения должны быть в исходном списке
