# Задание 29.4. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Вложенные структуры. Разбираемся, как хранить связанные данные в списке словарей и доставать конкретные элементы без путаницы.

## Условие задачи
Ты ведешь каталог настольных игр для клуба:
1. Создай список словарей с ключами `title`, `genre`, `players`, `tags` (список строк с особенностями).
2. Напечатай каталог в формате `Название (жанр). минимум игроков X`.
3. Собери список игр семейного жанра. фильтруй по значению `genre`.
4. Создай отдельный список всех уникальных тегов. пройдись по вложенным спискам `tags`.
5. Построй выборку игр, которые подходят компаниям от 4 игроков, и выведи их названия.

## Пример входных данных
```python
board_games = [
    {"title": "Семейный пикник", "genre": "family", "players": 4, "tags": ["кооператив", "короткая"]},
    {"title": "Гонки по пустыне", "genre": "strategy", "players": 2, "tags": ["соревнование", "карты"]},
    {"title": "Башня героев", "genre": "family", "players": 5, "tags": ["приключение", "детям"]}
]
```

## Пример выходных данных
```
Семейный пикник (family). минимум игроков 4
Гонки по пустыне (strategy). минимум игроков 2
Башня героев (family). минимум игроков 5
Семейные игры. ['Семейный пикник', 'Башня героев']
Все теги. {'детям', 'короткая', 'кооператив', 'карты', 'соревнование', 'приключение'}
Игры для 4+. ['Семейный пикник', 'Башня героев']
```

## Критерии успешного выполнения
- Список составлен из словарей с одинаковыми полями
- Печать каталога происходит через цикл без хардкода индексов
- Уникальные теги формируются автоматически
- Фильтры не изменяют исходный список `board_games`

## Подсказки
- Для накопления тегов удобно использовать множество
- Проверяй условие `players >= 4` перед добавлением в итоговый список
- Чтобы скопировать список тегов, используй `tags.copy()` при необходимости

## Частые ошибки и ограничения
- Не удаляй элементы из исходного списка во время перебора
- Следи, чтобы ключи словаря совпадали по написанию
- Не записывай результаты в глобальные переменные. храни их рядом с кодом задачи
- Не создавай отдельные списки по каждому жанру вручную. используйте фильтрацию
