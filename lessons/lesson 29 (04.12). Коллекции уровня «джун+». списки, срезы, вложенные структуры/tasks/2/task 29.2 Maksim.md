# Задание 29.2. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Индексация и срезы. Тренируем извлечение подсписков с разными границами и шагом без дополнительных циклов.

## Условие задачи
Ты анализируешь список времени забегов на тренировках. Нужно:
1. Показать стартовый прогресс — первые четыре результата.
2. Вывести последние три результата, используя отрицательный старт.
3. Собрать каждый второй результат, начиная с первого индекса.
4. Построить зеркальную копию списка, чтобы видеть динамику в обратном порядке.

## Пример входных данных
```python
runs = [32, 31, 30, 29, 28, 30, 27, 26, 25]
```

## Пример выходных данных
```
Стартовые попытки: [32, 31, 30, 29]
Финиш недели: [30, 27, 26, 25]
Каждый второй: [31, 29, 30, 26]
Инверсия: [25, 26, 27, 30, 28, 29, 30, 31, 32]
```

## Критерии успешного выполнения
- Каждая выборка сделана срезом
- Присутствуют примеры с положительными и отрицательными индексами
- Инверсия построена с шагом `-1`
- Вывод сопровождается понятными подписями

## Подсказки
- Для «первых n» достаточно указать `[:n]`
- Последние элементы берутся через `[-n:]`
- Каждый второй элемент выбирается шагом `2`
- Инвертировать список можно `runs[::-1]`

## Частые ошибки и ограничения
- Не изменяй исходный список
- Избегай шага 0 — он вызовет ошибку
- Проверь длину списка перед выборкой, чтобы не получить пустой результат
- Не путай включаемые и исключаемые границы `stop`
