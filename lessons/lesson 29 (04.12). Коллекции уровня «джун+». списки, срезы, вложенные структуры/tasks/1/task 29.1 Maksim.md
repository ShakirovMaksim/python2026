# Задание 29.1. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Знакомство со списками. Тренируем создание списков, работу с индексами и аккуратный перебор без ошибок выхода за границы.

## Условие задачи
Ты собираешь набор тренировок на неделю. Создай список из базовых упражнений, добавь рекомендации от тренера в середину списка, выведи количество элементов, покажи третий и предпоследний элементы. В конце распечатай расписание с порядковыми номерами, чтобы видеть очередность.

## Пример входных данных
```python
base_workout = ["кардио", "пресс", "растяжка"]
coach = ["планка", "йога"]
```

## Пример выходных данных
```
Всего упражнений: 5
Третье: растяжка
Предпоследнее: планка
1. кардио
2. пресс
3. растяжка
4. планка
5. йога
```

## Критерии успешного выполнения
- Список создается и дополняется без ручного копирования значений
- Используются индексы для выборки элементов из середины и конца
- Количество элементов считается через `len`
- Результирующий список выводится в читаемом виде с номерами

## Подсказки
- Чтобы вставить элементы в середину, используй `insert`
- Проверяй длину списка перед обращением к предпоследнему индексу `-2`
- В `enumerate` можно указать `start=1`, чтобы нумерация начиналась с единицы

## Частые ошибки и ограничения
- Не пытайся изменять список внутри цикла перебора, собери все изменения заранее
- Избегай обращения по индексу, если список пустой или слишком короткий
- Не выводи список в виде одной длинной строки, разбей по строкам
- Следи, чтобы элементы оставались строками, а не списками внутри списка
